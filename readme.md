# Infrastructure Cost & Report Generator

## Overview

This project is a platform that allows users to design their own IT infrastructure, automatically calculate costs, and generate a complete, presentation-ready report with a single click. The goal is to simplify infrastructure planning, cost visibility, and decision-making for technical and non-technical stakeholders.

## Visão Geral (Português)

Este projeto é uma plataforma que permite ao usuário montar sua própria infraestrutura de TI, calcular custos automaticamente e gerar, com um único clique, um relatório completo e pronto para apresentação à chefia. O objetivo é facilitar o planejamento de infraestrutura, a visibilidade de custos e a tomada de decisão.

## Key Features

* Interactive infrastructure builder
* Automatic cost estimation based on selected resources
* Consolidated financial and technical summary
* One-click generation of management-ready reports
* Clear visualization for decision-making

## Funcionalidades Principais

* Montagem interativa da infraestrutura
* Estimativa automática de custos com base nos recursos escolhidos
* Resumo técnico e financeiro consolidado
* Geração de relatório pronto para apresentação com um clique
* Visualização clara para apoio à tomada de decisão

## How It Works

1. The user selects and configures infrastructure components.
2. The platform calculates the estimated costs in real time.
3. With one click, a structured report is generated for presentation to leadership.

## Como Funciona

1. O usuário seleciona e configura os componentes da infraestrutura.
2. A plataforma calcula os custos estimados em tempo real.
3. Com um clique, é gerado um relatório estruturado para apresentação à liderança.

## Use Cases

* Infrastructure planning and budgeting
* Cost analysis for cloud and on-prem environments
* Executive and management presentations

## Casos de Uso

* Planejamento e orçamento de infraestrutura
* Análise de custos para ambientes cloud ou on-premises
* Apresentações executivas e gerenciais
